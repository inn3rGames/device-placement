# Device Placement

Audiotool challenge solution

## Live demo

https://device-placement.netlify.app/

## Local usage

Clone repository

```bash
git clone https://github.com/inn3rGames/device-placement.git
```

Install dependencies

```bash
cd device-placement
```

```bash
npm install
```

Start application

```bash
npm run start
```

Visit http://localhost:8080/

## Production

```bash
npm run build
```
